# Tailwind + Vite Project

This is a minimal Vite + Tailwind CSS starter scaffold created for you.

Quick commands:

```bash
# install (if you didn't let the script install dependencies)
npm install

# start dev server
npm run dev

# build for production
npm run build

# preview production build
npm run preview
```

Files of interest:
- `index.html` — main HTML file
- `src/main.js` — entry JS (imports the CSS)
- `src/main.css` — Tailwind entry CSS (@tailwind base; components; utilities;)
- `tailwind.config.cjs` — Tailwind config (content paths)
- `postcss.config.cjs` — PostCSS config

If anything fails with `npx tailwindcss`, you can run the Tailwind binary directly from `./node_modules/.bin/tailwindcss`.
